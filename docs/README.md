# Thu muc tai lieu do an tot nghiep

Thu muc nay dung de nop cac tai lieu van ban va file trinh bay cua do an.

## Cau truc nop

- `01-report/`: quyen do an tot nghiep dinh dang `.docx` va `.pdf`.
- `02-slides/`: slide bao ve do an tot nghiep dinh dang `.pptx`.
- `03-poster/`: poster gioi thieu do an kho giay A1, kich thuoc 594 x 841 mm, dinh dang `.pdf`.
- `04-user-guide/`: huong dan su dung chuong trinh va chay demo.
- `05-demo-video/`: video chay demo chuong trinh neu can nop kem.
